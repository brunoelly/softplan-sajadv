# Desafio Técnico Softplan

Tecnologias utilizadas:
  1.Java 11
  2.Springboot
  3.Postgresql
  4.Angular 10
  5.Swagger

Para executar a aplicação:

1. executar o Back-End na pasta raiz com o comando abaixo ou no eclipse:
mvn spring-boot:run

2. executar o Front-End na pasta raiz:
npm start
